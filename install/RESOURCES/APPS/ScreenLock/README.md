# What is ScreenLock.app

It allows run the screen locker `xtrlock` from a menu.

# Dependency

See install script details.

# Install

You must have superuser privileges to type your password
when asked.

Execute:
```
./install.sh
```
This will copy necessary dependency
then building and installing the application.

# Using

```
openapp ScreenLocker
```

Or run it from the `Applications` folder.

# Help

It is provided from the `Info/Help...` menu item.
