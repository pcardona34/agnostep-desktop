# What is Launcher

It displays the default Applications folder content.
It allows the user to type letters to filter the view.

# Install

You must have superuser privileges to type your password
when asked.

Execute:
```
./install.sh
```

# Using

```
openapp Launcher
```

Or run it from the `Applications` folder.
