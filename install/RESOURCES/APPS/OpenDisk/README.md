# What is OpenDisk.app

It allows to display the content of a mounted removable disk.
It allows to unmount that disk by dragging its folder icon onto the Recycler.
It provide help about removable disks, namely particular cases such UASP issue.

# Dependency

OpenDisk is efficient with wmudmount dockapp already installed.

# Install

You must have superuser privileges to type your password
when asked.

Execute:
```
./install.sh
```

# Using

```
openapp OpenDisk
```

Or run it from the `Applications` folder.

# Help

It is provided from the `Info/Help...` menu item.
