# What is Pass.app

It provides a Search and a New panels to get a password from this key search orto create a new password.

Use menu items:

```
Search...
New...
```

# Help

Read from menu item `Info/Help...` to get helpon the way to create and initiate your passwords-store. 
 
# How to install

```
./install.sh
```
