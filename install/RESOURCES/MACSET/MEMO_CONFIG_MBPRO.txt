/etc/default/keyboard (replace existing contents):

XKBMODEL="apple"
XKBLAYOUT="fr"
XKBVARIANT="mac"
XKBOPTIONS="altwin:swap_lalt_lwin"


Apply immediately:

sudo dpkg-reconfigure --frontend=noninteractive keyboard-configuration

Dans la console:
sudo setupcon -f

sudo udevadm trigger --subsystem-match=input --action=change

xinitrc/xsession

# French Mac Keyboard
setxkbmap -model apple -layout fr -variant mac -option altwin:swap_lalt_lwin

----

Intel par défaut:

Configure X to use Intel by default (create /etc/X11/xorg.conf.d/20-intel.conf):

Section "Device"
  Identifier "IntelGPU"
  Driver "intel"
  Option "TearFree" "true"
EndSection

----

PowerManagement

sudo apt install tlp tlp-rdw powertop acpi
sudo systemctl enable --now tlp

Recommended TLP conf tweaks (edit /etc/tlp.conf): set USB_AUTOSUSPEND=1 and SCHED_POWERSAVER if desired.

-----
Trackpad
sudo apt install xserver-xorg-input-libinput libinput-tools xinput

Create /etc/X11/xorg.conf.d/40-libinput.conf if missing:

Section "InputClass"
  Identifier "touchpad"
  MatchIsTouchpad "on"
  Driver "libinput"
  Option "Tapping" "True"
  Option "NaturalScrolling" "True"
  Option "DisableWhileTyping" "True"
EndSection

----

No backlight/brightness keys: use intel_backlight interface; adjust with:

ls /sys/class/backlightecho <value> | sudo tee /sys/class/backlight/intel_backlight/brightness
