# AGNoStep - agnostic GNUstep Desktop
===================================

## GNUstep and ported OPENSTEP Games

- Chess: a 3D Chess game
- Freecell: a card game
- GMastermind: a mastermind game
- GMines: a mine swipper game
- Gomoku: a TicTacToe game
- GShisen: a MahJong game
- Jigsaw: a puzzle game.
- LapisPuzzle: a Tetris like game
- NeXTGo: the Go game ported from OPENSTEP. Uses the GNUGo backend. With a nicer background from Ladder.
- Sudoku: a sudoku game

## Removed

- Ladder: a Go game: removed in favour of NeXTGo.
