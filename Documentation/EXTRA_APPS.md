# AGNoStep - agnostic GNUstep Desktop

## Extra applications

### How to install those apps?

1) Within a TTY console, use the main script to launch the **AgnostepManager**:

````
	./agnostep.sh
````

1-BIS) In the **Applications Folder**,  select and open **AgnostepManager.app**.

2) Then, from the main menu , choose the item: **Extra**.

3) In the **Extra Applications** window: select (check) the desired apps to install. Validate: press the button **OK**.

### List of the Extra applications available

- Cenon: Vector Drawing and Desktop Publishing
- Cynthiune: Music Player
- FlexiSheet: a Spreadsheet inspired by Quantrix
- Graphos: a simple Vectorial Drawing app
- Grr: RSS Reader
- LaternaMagica: Image collection Viewer
- NetSurf: Web Browser: poor javascript renderinf engine until now
- PikoPixel: Pixel Art Editor
- Player: a wrapper for VLC video player (from GSDE)
- PowerPaint: Bitmap Drawing
- PPC: an PowerPC Emulator
- Preview: Image Viewer
- PRICE: Image Manipulation program
- TalkSoup: the IRC client

> [!NOTE]
> **Weather.app** has been removed in favour of native **Meteo.app**.
