# AGNoStep - agnostic GNUstep Desktop

## Extra applications

### How to install those apps?

1) Use the main script to launch the **Applications Manager**:

````
	./agnostep.sh
````

2) Then, from the main menu , choose the item: **Extra**.

3) In the menu **Extra Applications**: select (check) the desired apps to install. Validate: press the button **OK**.

### List of the Extra applications available

- Affiche: Sticky Notes
- Calculator
- Cenon: Vector Drawing and Desktop Publishing
- Cynthiune: Music Player
- DictionaryReader: Dictionary Lookup for the English Users
- FlexiSheet: a Spreadsheet inspired by Quantrix
- FontManager
- FTP: a FTP client
- Graphos: a simple Vectorial Drawing app
- Grr: RSS Reader
- GSPdf: PS and PDF reader/viewer
- HelpViewer: '.help' reader (Help Files have a .help extension).
- ImageViewer: an Image Viewer (from GSDE) - See Preview alternative.
- Ink: Rich text (rtf and rtfd) editor. Alternative to TextEdit.
- Installer: an experimental app to install GNUstep apps as package
- LaternaMagica: Image collection Viewer
- Librarian: Books and Documents Organizing (from GSDE)
- Memory (experimental): a memory usage monitor
- NetSurf: Web Browser: poor javascript renderinf engine until now
- OpenUp: archive manager from GSDE (see Zipper alternative)
- PikoPixel: Pixel Art Editor
- Player: a wrapper for VLC video player (from GSDE)
- PowerPaint: Bitmap Drawing
- Preview: Image Viewer
- PPC: an PowerPC Emulator
- PRICE: Image Manipulation program
- ScanImage: Scanner (sane) client (from GSDE)
- ScreenShot: screen recorder: We should better use the tool provided by WindowMaker.
- StepSync: Syncing Folders
- TalkSoup: the IRC client
- VolumeControl: AlsaMixer compliant mixer
- Wheather: a Weather app
- Zipper: an archives handler (zip, tar.gz, ...)
