# Agnostep new release target

## New apps and tools

- [x] Meteo: new dockable GNUstep Weather app with Wttr.in API, temp and prediction. With a Prefs panel to set the location.
- [x] UpMem: new dockable GNUstep Uptime and Memory usage.
- [x] Updater:  complete GNUstep rewriting with badge and progress indicator...
- [x] Birthday: complete GNUstep rewriting with badge and Event Panel....
- [x] OpenDisk: mounting removable disk without GWorkspace Desktop background. A companion for wmudmount dockapp.
- [x] Launcher: open Application Panel.
- [x] AgnostepManager: embedded as a GNUstep app.
- [x] Dico: a French Lookup Dictionary and Service too from DVLF of the University of Chicago
- [x] Mixer: a simplified ALSA compliant mixer, with PCM only. Forked from VolumeControl.
- [x] Pass: a GNUstep interface to the Unix Password Manager (pass).
- [x] Printer: Improved Cups interface access.
- [x] SaveLink: improved code for the Internet Shortcuts Manager.
- [x] ScreenLock: a screen locker using xtrlock.
- [x] Sound: help and test about sound  setting.
