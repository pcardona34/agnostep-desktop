# AGNoStep - agnostic GNUstep Desktop

## Developer environment

Agnostep is not only a desktop, it also a **workshop** which provides a complete set of GNUstep development tools. If you are not yet familiar with Objective-C and GNUstep applications building, you should read documentation from the [GNUstep Developer](https://developer.gnustep.org/) page introduction.

Installation of Devel GNUstep Applications is available from the main **Applications Manager** menu:

````
	./agnostep.sh
````

Than select **Devel** item.

## Developer applications

- EasyDiff: graphical Diff Tool
- Emacs: Emacs editor built with GNUstep menus 
- **Gemas**: a specialized editor for objective-C, plist, GSmarkup...
- **Gorm**: an Interface Builder clone for GNUstep. Read [Gorm manual](https://mediawiki.gnustep.org/index.php/Gorm_Manual) to master it.
- GSMarkupBrowser: it is a Devel tool, but it was installed within libs-renaissance at stage 1 (Frameworks and Libs).
- GSMarkupLocalizableStrings: idem.
- **ProjectCenter**: it is a Project Builder clone for GNUstep. Read [Project Center manual](https://mediawiki.gnustep.org/index.php/ProjectCenter.app).
- Thematic: a Theme Editor for GNUstep

See also: the GNU Nano Editor in the [Wrappers](./WRAPPERS.md) section
 for the GNU Nano Editor
